Como remover um cliente já atendido da lista de espera do atendimento

Passo 1.

Dentro do menu do veterinário digite 1 e aperte enter para acessar a opção "Fila de atendimento":


Passo 2.

Assim que você apertar enter vai aparecer as seguintes opções:


digite 2 e aperte enter para acessar a opção "Atender a fila" como mostra a imagem acima.


Passo 3.

O programa irá mostrar os códigos dos clientes que estão na lista de espera com suas respectivas senhas e, o programa te dará a opção de informar o cógido do cliente que será atendido:


Passo 4.

Após digitar o código todas as informações do pet do cliente serão mostradas na tela:

Como mostra na tela acima você vai digitar o procedimento que foi feito e apertar enter


Passo 5.

Para encerrar, digite novamente o código do cliente que foi atendido para fazer a remoção do mesmo da fila de atendimento:
