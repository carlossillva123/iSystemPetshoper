Menu da recepção


Como ver os dados do cliente já cadastrado

Passo 1.

Dentro do menu da recepção digite 4 e aperte enter para acessar a opção "Dados do cliente":


Passo 2.

Assim que você apertar enter, o sistema pedirá para você digitar o código do cliente que você deseja conferir os dados. Caso o cliente não esteja cadastrado irá aparecer uma mensagem dizendo que o código foi inválido e voltará para o menu principal:


Assim que consultar os dados basta apertar enter que o programa irá para a próxima janela

Passo 3.

Após consultar os dados e apertar enter vai aparecer a opção de adicionar a fila de atendimento:


Caso você queira adicionar o cliente na fila basta digitar 1 como mostra a imagem acima. A seguinte tela será mostrada:



Digite 1 ou 2 para adicionar o cliente já cadastrado em uma das filas.
